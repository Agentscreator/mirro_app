## Color Coding

### Score Badges
- **Green** (85-100): Excellent quality
- **Blue** (75-84): Strong quality
- **Gray** (65-74): Good quality
- **Orange** (<65): Needs improvement

### Progress Bars
- **Purple-Blue Gradient**: All score metrics
- **Full Bar**: 100%
- **Empty Bar**: 0%

### Layout Borders
- **Purple Ring**: Selected layout
- **No Border**: Unselected layout
- **Hover Glow**: On hover

This visual guide shows exactly what users will see when interacting with the PyTorch layout scoring system.
